# 关注

## 获取指定用户关注状态

#### method

- get

#### uri

- /user/feed

#### args

- user_id 用户id

#### res

```json

{
    code:0,
    msg:'',
    format_msg:'',
    data:{
        is_feed:1, // 0为没有关注，1为已经关注
    }
}

```
