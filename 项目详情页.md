#### uri

- /project/detail

#### args

- id  项目id


#### res

    msg:'',
    format_msg:'',
    data:{
        name:'', //项目名称

## 获取项目项目收藏

#### method

- get

#### uri

- /project/collect

#### args

- id 项目id

#### res

```json
{
    code:0,
    msg:'',
    format_msg:'',
    data:{
        collect:0 //1为收藏 0为不收藏
    }

}
```
